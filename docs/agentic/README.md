# Agentic Ledgers

Persistent ledgers for `/agentic` runs live here.

Use one markdown ledger per run with a filename like:

```text
YYYY-MM-DD-<repo>-<target-slug>-ledger.md
```

Each ledger should capture:
- scope
- ranked themes and findings
- completed batches
- commit hashes
- blockers
- newly discovered follow-on work
- next recommended passes

Use the agent's native todo or plan tool as the active execution queue.
The ledger is the durable artifact that survives compaction, restarts, and handoff.
